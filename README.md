# 五子棋游戏 🎮

一个使用原生 JavaScript 和 Canvas 实现的在线五子棋游戏，支持双人对战、悔棋等功能。

## 项目简介

这是一个简洁优雅的五子棋游戏实现，采用面向对象的设计模式，代码结构清晰，易于理解和扩展。游戏提供了流畅的用户体验和美观的界面设计。

## 功能特点

- ✨ **完整的游戏逻辑** - 实现标准五子棋规则，支持黑白双方对战
- 🎯 **智能胜负判定** - 自动检测四个方向的五子连珠
- ↩️ **悔棋功能** - 支持回退上一步操作
- 🔄 **重新开始** - 一键重置游戏，开始新对局
- 🎨 **精美界面** - 渐变背景、立体棋子、流畅动画效果
- 📱 **响应式设计** - 适配不同屏幕尺寸

## 技术栈

- **HTML5 Canvas** - 绘制游戏棋盘和棋子
- **原生 JavaScript (ES6+)** - 游戏逻辑实现
- **CSS3** - 样式和动画效果
- **模块化设计** - 使用 ES6 模块和类

## 使用方法

1. 克隆或下载本仓库
2. 直接在浏览器中打开 `gomoku.html` 文件即可开始游戏
3. 无需任何额外的依赖或配置

## 游戏规则

- 黑棋先行，黑白双方轮流落子
- 在横、竖、斜任意方向形成连续五个同色棋子即为获胜
- 支持悔棋功能，可以撤销上一步操作

## 代码结构

项目采用面向对象设计，主要包含两个核心类：

- **Board 类** - 负责棋盘的渲染和状态管理
- **GameManager 类** - 负责游戏逻辑、玩家切换和事件处理
